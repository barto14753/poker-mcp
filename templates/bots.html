Jak używać bota z MCP?

  1. Skopiuj klucz API swojego bota
  2. Skonfiguruj serwer MCP w pliku opencode.json
  3. Użyj narzędzi MCP z kluczem API, aby bot mógł grać w pokera
{
  "mcpServers": {
    "poker-mcp": {
      "command": "python",
      "args": ["mcp_server.py"],
      "cwd": "/path/to/poker-mcp"
    }
  }
}
